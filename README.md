# Quokka

Since I dont feel like making actual docs, there are going to be 9 main library functions. with 19 built in keywords

The main keywords are:
if, then, else, try, except, finally, and, or, not, new, send, receive, usbin, usbout funct

## USB Management
attach, detach, status, device, route, reroute

## Data Transfer
transmit (wrapper for send/receive), timeout, checksum, hash, sign, verify, sync, async, await

## Buffering/Queue
buffer, queue, push, pop, shift, unshift, allocate, free, malloc, dispose

## Logging/Debugging
log, debug, trace, observe, profile, stats

## SYS Management
configure, setup, reset, restart, watchdog

## Security
authenticate, authorize

## Data Transformation
compress, decompress, encode, decode

## Concurrency
lock, unlock, mutex, daemon, thread, task

## Event Handling
onconnect, ondisconnect, onerror, onreceive
